  • 究極能力 正義之王　/　ミカエル
    • 究極能力天使系美徳系支配系
    • 権能
      王権王宮城塞究極付与天使之軍勢他
    • 所有者
      ルドラ・ナスカ フェルドウェイ

    創造主ヴェルダナーヴァが生み出した権能、天使系。その上位である“七つの美徳”のひとつ。
    “星王竜”ヴェルダナーヴァが創造した最強の究極能力。その命令は精神支配そのものであり、“王宮城塞”のような本物の『絶対防御』を有する、指揮系統に特化した権能。意志ある者すべてを支配し、被支配者の権能を間借りして扱うことまで出来る、王者として君臨するに相応しいであり、ルドラが皇帝として世界を統一するための後押しとして、ヴェルダナーヴァが『誓約之王』との交換で与えた。

    “王権発動（レガリアドミニオン）”　…
    絶対的な支配の権能。『正義之王』の真骨頂。対象に自我まで奪う強制支配を行う。被支配者の存在値の合計が権能主の存在値を超えるまで複数人を支配することが出来る。またその性質上、同格が相手では成功率が下がり、最低でも自分を下回った状態でなければ成功しない。

    “支配の弾丸（ドミニオンブレット）”　…
    近藤中尉に貸し与えられていた支配の権能。“王権発動”と異なり対象は一人、さらに相手を油断させなければ成功率も低いというものだが、近藤は過去にクレイマンとカガリを支配することに成功している。

    “天使長の支配（アルティメットドミニオン）”　…
    “王権発動”と対を成す支配の権能。天使系究極能力保有者への絶対支配を行う。ヴェルダナーヴァが構築した天使系と『正義之王』の間に繋がる極細の“支配回路”を通して、対象に支配された自覚すら抱かせない絶対支配を行い、自由に命令を与えることができるようになる。支配をかける条件は、対象と視線を交差させること。

    “王宮城塞（キャッスルガード）”　…
    あらゆる攻撃を無効化する本物の『絶対防御』。常時発動型の完璧な防御ではあるが、利用には権能主への忠誠心がエネルギー源となるため権能主に忠実な者がいなければならないという条件と、発動中は魔法や他の権能を含む攻撃手段が取れず、覇気や武器による攻撃しか出来なくなるという制限もある。
    ヴェルダナーヴァからこの権能を預かっていたルドラは極めた技量で、『斬撃の瞬間だけ、“王宮城塞”を解除し、闘気を通して威力を増す』といった変則技まで実現させていた。

    『究極付与（アルティメットエンチャント）』　…
    自分の権能を他者に貸し与える権能。覚醒していない者を簡易的に究極の領域に引き上げることができ、最低限ではあるが上位者と戦う土俵に立つ資格を与えることが出来る。貸し与えられるのは受け皿となる肉体の強度にもよるが、基本的に劣化した権能だけであり、また付与されたものであるため奪われやすい。

    『天使之軍勢（ハルマゲドン）』　…
    天使族の軍勢を召喚して意のままに操ることができる権能。一度に召喚できる天使族の数に制限はないが、エネルギーの総量には限界があり、無数の天使を召喚して軍勢として運用する他、エネルギーを集中させて数柱の熾天使を創り出すことも可能。また召喚された天使族には肉体がないため、長くても一週間足らずで消滅する。

  • 究極能力 誓約之王　/　ウリエル

    創造主ヴェルダナーヴァが生み出した権能、天使系。その上位である“七つの美徳”のひとつ。
    ナスカ王国の王太子にして“始まりの勇者”であるルドラに発現した、ルドラの信念や世界を統一するという誓い、それに応える仲間達の想いの結晶。空間管理に特化しており、あらゆる事象・権能を管理し把握するための権能。ルドラに宿った『誓約之王』は“破邪”と“守護”の権能だった。また、ヴェルダナーヴァの本来の『誓約之王』には民の声を聞く作用があり、ルドラはそれを利用することによる、配下の忠誠心を糧にした防御を行う『正義之王』と同質のエネルギーを纏った攻撃が可能だった。
    ヴェルダナーヴァがルドラとこの権能を交換したまま死亡したことで、権能は輪廻の輪に取り込まれたが、後にリムルが『無限牢獄』をベースに様々な権能を集約させることで獲得に成功した。曰く、配下からの忠誠の結晶。

    『無限牢獄（インフィニティプリズン）』　…
    対象を虚数空間へ閉じ込める牢獄。かつてヴェルドラを封印していた勇者クロノアの結界と同等の性能を誇る。

    『法則操作』　…
    黒炎雷。魔力操作。熱操作及び、慣性制御。自在に『胃袋』への熱の出し入れを可能とする。魔素を操ることで様々な事象改変を行う。全属性の『法則操作』により『天邪鬼』の再現も可能。

    『空間支配』　…
    移動能力。位置座標を認識した空間を自在に入れ替えることで、瞬間移動に近い移動を可能にする。『万能感知』で認識する範囲内ならば、空間に穴を開けなくても一瞬で転移可能であり、少し時間はかかるが、一度行った遠方の場所にも行ける。

    『万能結界』　…
    多重構成された複合結界と、空間断絶による絶対防御。『智慧之王』の支配下にあり、自動でリムルの身体を護っている。

    『絶対防御（アブソリュートエンド）』　…
    ルドラが『無限牢獄』を応用した、肝心なときには破られてしまう権能。これは恐らく“霊子”の不規則な動きを捉えられていないからであり、リムルは『智慧之王』によりこの乱数位相の認識に成功し、聖属性攻撃の予測防御も含めた『絶対防御』として運用させるに至った。

    『絶対切断（アブソリュートエンド）』　…
    ルドラが『絶対防御』を転用して編み出した攻撃手段。『誓約之王』の民の声を聞く作用を利用することによって、信じる者の数だけ“力”が増す。同様のエネルギーを纏う『正義之王』の“王宮城塞”を打ち破る可能性を持つ数少ない攻撃手段のひとつ。リムルの『誓約之王』にこの作用が残っているかは不明だが、ルドラと同様に、『絶対防御』を反転させることで『絶対切断』を再現した。
